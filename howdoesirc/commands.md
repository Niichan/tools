Basic commands:

```
/join #channel 	- joins you to #channel
/nick newnick 	- changes your nick to newnick
/part #channel 	- parts you from a channel
/me action 		- makes you do an action
/list 			- shows a list of channels in your status buffer.
/away reason 	- marks your client as away with a reason 
/whois name 	- Looks up information on a user, list of publically visible channels joined, away status
/topic 			- Lets you see or manipulate the channel topic
/me action		- has you perform an action
```

Channel operator commands:

```
/mode [Arguments] 		- See /quote help cmode
/kick user [reason] 	- kicks a user with an optional reason
```
